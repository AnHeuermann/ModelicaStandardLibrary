visc_dT

Dynamic viscosity eta(d,T), industrial formulation

Syntax

eta = visc_dT(d, T, p, phase, checkLimits)

Inputs (5)

d

Type: Density (kg/m³)

Description: Density
T

Type: Temperature (K)

Description: Temperature (K)
p

Default Value: 0

Type: Pressure (Pa)

Description: Pressure (only needed for region of validity)
phase

Default Value: 0

Type: Integer

Description: 2 for two-phase, 1 for one-phase, 0 if not known (unused)
checkLimits

Default Value: false

Type: Boolean

Description: Check if inputs d,T,P are in region of validity

Outputs (1)

eta

Type: DynamicViscosity (Pa·s)

Description: Dynamic viscosity
