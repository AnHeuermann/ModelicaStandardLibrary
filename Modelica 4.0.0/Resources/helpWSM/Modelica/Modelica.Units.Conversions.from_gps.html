from_gps

Convert from gram per second to kilogram per second

Syntax

kgps = from_gps(gps)

Inputs (1)

gps

Type: MassFlowRate_gps (g/s)

Description: Value in g/s

Outputs (1)

kgps

Type: MassFlowRate (kg/s)

Description: Value in kg/s
