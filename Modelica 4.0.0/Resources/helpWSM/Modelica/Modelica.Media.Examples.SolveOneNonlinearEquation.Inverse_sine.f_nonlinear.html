f_nonlinear

Define sine as non-linear equation to be solved

Syntax

y = f_nonlinear(u, A, w, s)

Inputs (4)

u

Type: Real

Description: Independent variable
A

Default Value: 1

Type: Real

Description: Amplitude of sine
w

Default Value: 1

Type: Real

Description: Angular frequency of sine
s

Default Value: 0

Type: Real

Description: Shift of sine

Outputs (1)

y

Type: Real

Description: Dependent variable y=f(u)
