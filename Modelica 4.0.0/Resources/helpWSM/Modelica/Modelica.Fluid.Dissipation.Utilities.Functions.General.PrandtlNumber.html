PrandtlNumber

Calculation of Prandtl number

Syntax

Pr = PrandtlNumber(cp, eta, lambda)

Inputs (3)

cp

Type: SpecificHeatCapacityAtConstantPressure (J/(kg·K))

Description: Specific heat capacity of fluid at constant pressure
eta

Type: DynamicViscosity (Pa·s)

Description: Dynamic viscosity of fluid
lambda

Type: ThermalConductivity (W/(m·K))

Description: Thermal conductivity of fluid

Outputs (1)

Pr

Type: PrandtlNumber

Description: Prandtl number
