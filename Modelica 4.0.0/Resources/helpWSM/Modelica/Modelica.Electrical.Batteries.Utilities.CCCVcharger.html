CCCVcharger

Charger with constant current - constant voltage characteristic

Information

This information is part of the Modelica Standard Library maintained by the Modelica Association.

Ideal charger switching from constant current to constant voltage characteristic.

Parameters (4)

I

Value:

Type: Current (A)

Description: Constant charge current
startTime

Value: 0

Type: Time (s)

Description: Start time of charging
rampTime

Value: 60

Type: Time (s)

Description: Ramp up charging current
Vend

Value:

Type: Voltage (V)

Description: End of charge voltage

Type: PositivePin

Description: Positive electrical pin

Type: NegativePin

Description: Negative electrical pin
