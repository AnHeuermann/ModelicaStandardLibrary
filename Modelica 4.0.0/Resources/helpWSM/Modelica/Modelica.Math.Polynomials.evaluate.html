evaluate

Evaluate polynomial at a given abscissa value

Syntax

y = evaluate(p, u)

Inputs (2)

p

Type: Real[:]

Description: Polynomial coefficients (p[1] is coefficient of highest power)
u

Type: Real

Description: Abscissa value

Outputs (1)

y

Type: Real

Description: Value of polynomial at u
