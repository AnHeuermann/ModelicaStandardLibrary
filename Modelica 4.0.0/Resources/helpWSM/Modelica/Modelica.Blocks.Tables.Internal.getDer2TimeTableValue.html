getDer2TimeTableValue

Second derivative of interpolated 1-dim. table where first column is time

Syntax

der2_y = getDer2TimeTableValue(tableID, icol, timeIn, der_timeIn, der2_timeIn)

Inputs (5)

tableID

Type: ExternalCombiTimeTable

Description: External table object
icol

Type: Integer

Description: Column number
timeIn

Type: Real

Description: (Scaled) time value
der_timeIn

Type: Real

Description: Derivative of (scaled) time value
der2_timeIn

Type: Real

Description: Second derivative of (scaled) time value

Outputs (1)

der2_y

Type: Real

Description: Second derivative of interpolated value
